# Annapurneshwari Caterers Website

A complete catering website built for a real client to simplify menu selection, booking, and communication.

## 🌐 Live Website

https://annapurneshwaricaterers.netlify.app/

## 🚀 Features

* Interactive menu selection from full catering catalog
* Add dishes to cart
* Drag & reorder selected items
* Event details input (name, phone, location, date, time)
* Google Maps location link support
* WhatsApp integration for sending menu directly
* PDF generation of selected menu
* Clean and responsive UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Netlify (Deployment)
